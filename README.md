# LLM Sentiment Chatbot

A full-stack sentiment analysis chatbot built using Flask and OpenAI LLMs.

## Features
- Chatbot-style UI
- Real-time sentiment analysis
- Uses OpenAI LLM (gpt-4o-mini)
- Secure API key handling using dotenv

## Tech Stack
- Python
- Flask
- OpenAI API
- HTML, CSS, JavaScript
- python-dotenv
